# Money Bank - Мобильное банковское приложение

## Быстрый старт

- Ссылка на приложение - `money-bank-m.vercel.app`
- Перед запуском необходимо инициализировать проект с использованием React/Vite - `npm install`
- Запуск проекта осуществляется командой: `npm run dev`

## Возможности приложения:

**Банковские функции:**

- Обмен валют - покупка/продажа с реальными курсами ЦБ РФ
- Курсы валют - актуальные данные с API Центробанка
- Виртуальный баланс - управление счетами в разных валютах
- Банк на карте

**Пользовательский опыт:**

- Авторизация - форма регистрации и входа
- Темная тема - переключение светлой/темной темы
- Поддержка русского и английского языка
- Интерфейс для мобильных устройств
- LocalStorage - сохранение данных пользователя

**Интеграции:**

- API Центробанка РФ - получение курсов валют
- Картографические сервисы - Yandex Maps / 2GIS

## Технологический стек:

- React 18 + TypeScript
- React Router v6 (навигация)
- Redux Toolkit (управление состоянием)
- React-i18next - интернационализация (RU/EN)
- Yandex Maps - карты
- CSS Modules + CSS Variables - стилизация
- Vite (сборка проекта)

## Примечание:

- Проект представляет собой прототип мобильного банковского приложения
- Основная идея: создание виртуальной песочницы для оттачивания навыков программирования, что позволяет:
  освоить полный цикл разработки приложения.
- Ключевое преимущество: приложение обладает масштабируемостью, что позволяет легко вносить изменения, добавлять улучшения и реализовывать различные функции, используемые в реальных веб-приложениях.
